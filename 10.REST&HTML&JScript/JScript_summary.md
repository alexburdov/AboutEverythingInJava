#### TypeScript
TypeScript — это
язык программирования от Microsoft, надстройка над JavaScript, добавляющая статическую типизацию и другие возможности для создания надежных, масштабируемых приложений и облегчения работы с большими проектами, компилируемый в обычный JavaScript для выполнения в браузерах.
