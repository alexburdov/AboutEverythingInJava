#### Inner Class
Внутренний класс (inner class) в Java — это
класс, объявленный внутри другого класса, который позволяет логически сгруппировать связанные классы, скрывает их и дает внутреннему классу прямой доступ ко всем (включая private) полям и методам внешнего класса. Они связаны с экземпляром внешнего класса, в отличие от статических вложенных классов (static nested classes), которым нужен экземпляр только для доступа к членам.

**Основные характеристики**

- **Связь с внешним классом:** Нестатические внутренние классы неразрывно связаны с экземпляром внешнего класса и создаются только через его объект, например, Outer.Inner inner = outer.new Inner();.
- **Доступ к членам:** Имеют полный доступ ко всем членам (полям, методам) внешнего класса, даже если они private.
- **Модификаторы доступа:** Могут иметь любые модификаторы доступа (public, protected, private, default), что позволяет скрывать их от остального кода.
- **Помощь в модульности:** Улучшают организацию кода, группируя вспомогательные классы, которые нужны только в контексте одного внешнего класса

**Типы внутренних классов**

- **Нестатический внутренний класс (Inner Class):** Классический внутренний класс, связанный с экземпляром.
- **Локальный класс (Local Class):** Объявляется внутри блока кода (метода, цикла, if), связан с экземпляром и доступен только в этом блоке.
- **Анонимный класс (Anonymous Class):** Локальный класс без имени, используется для одноразового создания и инициализации, часто для переопределения методов интерфейсов/классов.
- **Статический вложенный класс (Static Nested Class):** Похож на обычный класс, но статичен и не связан с экземпляром внешнего класса. Для создания не нужен экземпляр внешнего класса
